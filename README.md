## Description

This is a starter project for a scalable backend service. It works particularly with TypeORM, PostgreSQL database and other libraries like class-validator and class-transformer for secure input and output.

## Features added

#### Application configuration - [Completed]

NODE_ENV=value should match with .env.value
You can import AppConfigService in any module and use necessary config data.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
